title Sequence Diagram: 14 Modify Event

actor "Organiser" as organizer
control "<link:_>: EventController" as eventController
participant "<link:_>: EventService" as eventService
entity "<link:_>e: Event" as event

==1. The organiser selects the edit event functionality\n2. The system displays the add and modify event page (19-modify-event-organiser) page\n3. The organiser modifies the details of the event==
organizer -> eventController: modifyEvent(details)
activate eventController
eventController -> eventService: modifyEvent(details)
activate eventService
eventService -> eventService: getOrganiser(details)
activate eventService
eventService --> eventService: return o
deactivate eventService
note over eventService: The getOrganiser method is a standin for\nthe future operation of retrieving the organiser\n with the correct details from database

eventService -> event: updateEvent(details)
activate event
note over eventService: The updateEvent method modifies the existing event\nin the database with the provided updated details.

event --> eventService:displayEvent(details)
deactivate event

eventService --> eventController: displayEvent(details)
deactivate eventService

eventController --> organizer: displayEventSchedule("20-events-organiser")
deactivate eventController
