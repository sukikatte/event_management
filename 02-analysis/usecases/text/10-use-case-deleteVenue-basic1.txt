actor Administrator
control ":VenueController" as VenueController

== 1. The administrator selects the delete venue functionality adjacent to the venue they want to remove on the list venues page ==

Administrator -> VenueController: deleteVenue()

VenueController -> VenueService: getVenue(name)

note over VenueController: The checkIfVenueExists method checks \n if the venue exists before deletion

VenueService --> VenueController: return TRUE

VenueController -> VenueService: deleteVenue()
activate VenueService

note over VenueController: The deleteVenue method deletes the selected \n venue from database if the venue exists

VenueService --> VenueController: return NONE
deactivate VenueService

note right of VenueController: This part happens in the JavaScript.

VenueController -> Administrator: return delete warning

== 2. The system asks the administrator to confirm the deletion of the venue (13-delete-venue-warning) ==

VenueController -> Administrator: return to the list venues page (17-list-venues-admin)
